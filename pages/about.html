Pixlit 소개

빠르고, 무료이며, 완전히 안전한 이미지 도구

만든 이유

아이폰으로 찍은 사진을 공유하거나 웹에 올리려면 HEIC 파일을 JPG로 변환해야 합니다. 기존 도구들은 파일을 서버에 업로드해야 했고, 용량 제한이 있거나 구독료를 요구했습니다. Pixlit은 이 문제를 브라우저 기술로 완전히 해결했습니다.

작동 방식

Pixlit의 모든 이미지 처리는 사용자의 브라우저 안에서만 이루어집니다. 파일은 절대 외부 서버로 전송되지 않습니다. 이는 다음을 의미합니다:

  • 인터넷 연결 속도에 관계없이 빠른 처리
  • 파일 크기 제한 없음
  • 업로드로 인한 개인정보 유출 위험 없음
  • 서버 운영 비용 없이 완전 무료 서비스 가능

제공 기능

  • HEIC → JPG 변환 — 아이폰 사진을 어디서든 열 수 있는 JPG로 변환
  • 이미지 압축 — PNG, JPG, WEBP 파일을 품질을 유지하면서 용량 축소
  • 일괄 처리 — 여러 파일을 한 번에 처리하고 ZIP으로 다운로드
  • Before / After 미리보기 — 변환 결과를 즉시 확인

기술 스택

순수 HTML, CSS, JavaScript로 제작되었습니다. 외부 서버 없이 브라우저의 Canvas API와 WebAssembly를 활용해 고성능 이미지 처리를 구현했습니다.
